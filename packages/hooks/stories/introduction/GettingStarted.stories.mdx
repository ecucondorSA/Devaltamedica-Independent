# 🏥 @altamedica/hooks

**React hooks centralizados para la plataforma AltaMedica**

Biblioteca completa de hooks especializados para aplicaciones médicas con compliance HIPAA, funcionalidades de telemedicina, IA médica y performance optimizado.

## 🚀 Instalación

```bash
# NPM
npm install @altamedica/hooks

# Yarn
yarn add @altamedica/hooks

# PNPM
pnpm add @altamedica/hooks
```

## 🎯 Ejemplo Rápido

```typescript
import React from 'react';
import { usePatients, useAuth } from '@altamedica/hooks';

function MedicalDashboard() {
  // Autenticación con permisos médicos
  const { user, hasPermission } = useAuth();
  
  // Gestión de pacientes con compliance HIPAA
  const { 
    patients, 
    isLoading, 
    search,
    createPatient 
  } = usePatients({
    hipaaCompliant: true,
    auditLog: true
  });

  if (!hasPermission('read_medical_records')) {
    return <div>Sin permisos para ver registros médicos</div>;
  }

  return (
    <div>
      <h1>Dashboard Médico - Dr. {user?.name}</h1>
      
      <input 
        placeholder="Buscar pacientes..."
        onChange={(e) => search(e.target.value)}
      />
      
      {isLoading ? (
        <div>Cargando pacientes...</div>
      ) : (
        patients.map(patient => (
          <div key={patient.id}>
            {patient.name} - {patient.priority}
          </div>
        ))
      )}
    </div>
  );
}
```

## 🏥 Características Médicas Especiales

### ✅ Compliance HIPAA
- Encriptación automática de PHI (Protected Health Information)
- Audit trails para todas las operaciones médicas
- Validación de permisos granulares por rol médico

### ✅ Telemedicina Avanzada
- WebRTC optimizado para videollamadas médicas
- Latencia ultra-baja para consultas críticas
- Grabación segura con compliance médico

### ✅ IA Médica Integrada
- Análisis de síntomas con machine learning
- Sugerencias diagnósticas basadas en evidencia
- Detección automática de emergencias médicas

### ✅ Performance Médico-Específico
- Monitoreo de operaciones médicas críticas
- Alertas de performance para sistemas hospitalarios
- Optimización para workstations médicas

## 🔒 Seguridad y Compliance

### HIPAA Compliance Built-in
- ✅ Encriptación AES-256 para datos PHI
- ✅ Audit logs automáticos con timestamps
- ✅ Validación de acceso basada en roles
- ✅ Anonimización automática de datos sensibles
